PROJETOS

ART DIALOGUE

Incrível projeto feito usando Ruby on Rails, durante o Le Wagon, onde eu tive o prazer de produzir com uma equipe de 5 pessoas. O projeto consiste em uma rede social para entusiastas de filmes, livros e músicas, com sistema de seguidores e sistema de avaliações.

CAÇA AO TESOURO

Projeto de Caça ao Tesouro feito usando Python, fizemos esse jogo como pedido de AP2 do curso de Ciência de Dados na IBMEC. O projeto consiste em um jogo onde o jogador se movimenta em um mapa e ao longo do caminho ele se depara com monstros e precisa lutar. O jogo acaba quando o personagem morre ou chega com sucesso ao tesouro.

PROJETO DOSIMAGEM

Projeto de Front-end para a empresa DosImagem feito usando React, fizemos esse projeto ao longo do nosso semestre do curso de Ciência de Dados na IBMEC. O projeto consiste em um aprimoramento do front-end da API usada pela empresa, para que o funcionário responsavel pelas ações do sistema não precise receber instruções de como usar o sistema, mas sim, usando-o intuitivamente.
